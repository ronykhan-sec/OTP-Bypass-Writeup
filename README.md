# Real-World OTP Bypass Vulnerabilities

## Overview
Discovered OTP bypass vulnerabilities during security testing.

## Vulnerabilities

### 1. Sign-Up OTP Bypass
OTP verification could be bypassed during registration.

Impact:
- Create accounts without verification

### 2. Login OTP Bypass
OTP verification could be bypassed during login.

Impact:
- Access accounts without OTP

## Lessons Learned
- Authentication testing
- OTP workflows
- Request manipulation
